About Me

Hello, I'm Nguyen Hoang Thien, a Unity Game Developer focused on developing 2D mobile games.

I have experience working with Unity and C#, especially in gameplay programming, UI systems, player data management, advertisements, and in-app purchases.

During my learning and project development process, I have gained knowledge of OOP, SOLID principles, and Design Patterns such as Singleton, Observer, Object Pooling, and State Machine.

I always aim to build clean, scalable, and maintainable game systems. My goal is to become a professional Unity Developer and contribute to high-quality commercial game projects.

Besides programming, I am also interested in player experience, gameplay feel, visual feedback, and making games smoother, more understandable, and more enjoyable.

Stop Move Stop

Stop Move Stop is my first 3D mobile game developed using Unity and C#. The project focuses on gameplay programming, character control, obstacle systems, and user interface implementation.

During the development process, I gained practical experience in building 3D gameplay mechanics, handling player interactions, managing game states, and designing reusable game systems. This project also helped me improve my code organization and problem-solving skills.

The main technologies used in this project include Unity Engine, C#, Unity UI, and JSON-based data management. Through this project, I strengthened my understanding of OOP principles and component-based development in Unity.

Google Play GitHub

Cuoc Chien Quai Vat

Cuoc Chien Quai Vat is a 2D mobile game developed and published on Google Play using Unity and C#. This project was independently developed from start to finish, including game design, gameplay programming, user interface implementation, audio integration, testing, and deployment.

The game features character purchasing, upgrading, and merge mechanics, along with progression systems, popup management, tutorial guidance, and player data handling. Through this project, I gained valuable experience in designing scalable game systems and creating a complete production workflow.

The project includes Google AdMob advertisement integration and In-App Purchases (IAP) for monetization. The main technologies used are Unity Engine, C#, Scriptable Objects, JSON data management, Unity UI, and design patterns such as Singleton, Observer, and Object Pooling.
